# 🩸 blood-donor-note

> 2026 헌혈 공모전 굿즈 부문 출품을 목적으로 개발된 프로젝트입니다. <br/>
> Developed for submission to the 2026 Blood Donation Contest (Goods Category)

<br/>

### Project Info (프로젝트 정보)

 NFC 키링 태그를 통해 웹페이지에 간편하게 접속하고, 개인의 헌혈 정보를 확인할 수 있는 웹 서비스입니다.
사용자는 키링 태그를 통해 자신의 헌혈 이력을 기록하고, 헌혈 횟수, 다음 헌혈 가능일, 누적 정보 등 다양한 데이터를 확인할 수 있습니다.
또한 기록 기반 통계 시각화, 헌혈 주기 안내, 참여를 유도하는 인터랙션 요소 등을 제공하여, 단순 정보 확인을 넘어 지속적인 헌혈 참여를 지원합니다.
별도의 앱 설치 없이 접근 가능한 구조로 설계되어 일상 속에서 쉽고 빠르게 활용할 수 있습니다.
